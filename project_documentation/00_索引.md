## 项目概述

Cherry Studio是一个功能强大的AI助手应用，主要特点包括：

- 支持多种AI模型和提供者（OpenAI、Gemini、Anthropic等）
- 内置300+预配置AI助手
- 知识库管理和检索
- 工作区文件管理
- 翻译功能
- AI绘画功能
- 文件管理和处理
- 跨平台支持（Windows、Mac、Linux）

## 技术栈

- Electron - 跨平台桌面应用框架
- React - 前端UI库
- TypeScript - 类型安全的JavaScript超集
- Redux - 状态管理
- Dexie - IndexedDB封装库
- Ant Design - UI组件库
- Styled Components - CSS-in-JS解决方案
